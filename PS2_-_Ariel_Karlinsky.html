Q3

As we will only use quantitative variables, we create a new trimmed down version of the dataset without the following variables: ID, Gender, Student, Married, Ethnicity and store it as Dat2. We then load this trimmed down version for glmnet with model.matrix() (we remove the intercept using -1 in the model.matrix() formula:

Dat2 <- Dat %>% 
  select(-c(ID,Gender,Student,Married,Ethnicity))
model_matrix <- model.matrix(object = Income ~. - 1, data = Dat2)

Q4

All the models we estimate have the following form, they minimize the following problem, as seen here and in the “Dimensionality Reduction slides”

\[ \min_{\beta} \frac{1}{N} \sum_{i=1}^{N}(y_i-\beta^T x_i) + \lambda\left[(1-\alpha)||\beta||_2^2/2 + \alpha ||\beta||_1\right] \ \]

The ridge model is given by \(\alpha=0\) (as we are left with the L2 norm of \(\beta\)), The lasso model by \(\alpha=1\) and the elastic net model is a combination of the two and is thus given by \(\alpha \in (0,1)\). $is a the regularization parameter that determines how much weight is given to the penalty. The models are estimated with the glmnet package, where we estimate 2 elastic-net models - one in which \(\alpha\) is close to 0 and one in which it’s close to 1.

fit_lasso <- glmnet(y=Dat2$Income, x=model_matrix, alpha = 1)
fit_ridge <- glmnet(y=Dat2$Income, x=model_matrix, alpha = 0)
fit_elasticnet1 <- glmnet(y=Dat2$Income, x=model_matrix, alpha = 0.05)
fit_elasticnet2 <- glmnet(y=Dat2$Income, x=model_matrix, alpha = 0.95)
